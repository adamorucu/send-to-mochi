# Send to Mochi

A plugin sync flashcards created in Obisidian with mochi.

## Features
- Allows for different card structures - using a separator (`---`) or cloze (`{{1::text}}`).
- adds a unique id for each card locally
- uploads newly created cards using the mochi API
- updates cards contents if they have been changed in obsidian.
